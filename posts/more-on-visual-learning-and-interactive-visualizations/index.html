In a previous post, I compiled some examples and resources on interactive visualizations. In this post, I am listing a few more that are quite interesting and useful.


Explorable Explanations


I am a fan of explorable explanations, the revamped site includes tools and tutorials to make your own explorables. Check out Nicky Case's post on the thought process behind creating an explorable explanation and Bret Victor's article on what it means to be an active reader. I recently came across Chalktalk which is like an interactive blackboard for exploring solutions by collaborating with the audience during presentations, read this article for more information.


Machine Learning


Lately, there has been a lot of discussion around interpretability and explaining model predictions. There has also been a rise in the number of tools and frameworks that can help in explaining model predictions through visualization or other methods. LIME seems to be a popular framework especially used by data scientists in industry. Distill has an article on feature visualization to explain how neural networks build up their understanding of images. RNNVis is a recent project that helps visualize the hidden states of recurrent neural networks, this project is under active development. The studio otoro blog is great for generative design and machine learning research articles with lots of visual examples.


Side Note: interesting stuff from Singapore


If you are in Singapore, it's likely you have heard about the mysterious Circle Line fault that was all over the news. Data.gov.sg has a detailed post on how this issue was solved which includes the use of a not-so-common Marey chart. Also, check out this really cool visualization using isochrone map of Singapore to show public transport accessibility. If you want to know more about isochrone maps, you can read Yin Shanyang's post.


To wrap-up this post, here's a comprehensive visualization catalog with code templates: The Python Graph Gallery and here's something for inspiration: Dear Data Project.
